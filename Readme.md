# Churn Prediction

This project is an end-to-end machine learning system that predicts customer churn and suggests retention actions.

## 🚀 Live Demo


## 📊 Problem
Businesses lose revenue when customers churn. This project predicts churn probability and recommends actions.

## 🧠 Solution
- Feature engineering based on user behavior
- XGBoost model for prediction
- Threshold-based decision system

## ⚙️ Tech Stack
- Python
- Pandas, Scikit-learn
- XGBoost
- Streamlit

## 🔥 Key Results
- ROC-AUC: 0.93
- Recall (churn): 84%
- Real-time prediction system deployed

## 💡 Features
- Single user prediction
- Batch CSV prediction
- Business action recommendations
